Installation, Version, and Database Requirements

    Software Requirements:
        Microsoft Visual Studio
            Version: 14.0.25431.01 Update 3
        Microsoft .NET Framework
            Version 4.6.01586
        Microsoft.AspNet.Mvc
            Version 5.2.3
        Entity Framework
            Version 6.1.3
            
    Database:
        AdventureWorks Databases – 2014 mdf from https://msftdbprodsamples.codeplex.com/ file is needs to be placed in the App_Data folder before attempting to run this program.
