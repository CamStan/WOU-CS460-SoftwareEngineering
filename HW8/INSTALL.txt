Installation, Version, and Database Requirements

    Software Requirements:
        Microsoft Visual Studio
            Version: 14.0.25431.01 Update 3
        Microsoft .NET Framework
            Version 4.6.01586
        Microsoft.AspNet.Mvc
            Version 5.2.3
        Entity Framework
            Version 6.1.3
            
    Database:
        Before attempting to run this application, the proper database is needed
            - Create a new, empty, database to be stored in the App_Data folder
            - Open a new query from that database
            - Run the initDB.sql code inside that query to create the tables
            - Run the populate.sql code inside another query to populate the tables
